# RDA5807_FM_Radio
 - RDA5807を使用したFMラジオです。
 - 選曲はロータリーエンコーダで選択。
 - 周波数＋ 局名表示はOLED（有機EL）ディスプレイ
 - 地上波の周波数と、CATV（JCOM千葉）のFM再送信周波数を切り替えて選択します。
 - 手動チューニングは100KHzステップ
